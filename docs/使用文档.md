# OpenClaw Dashboard Pro 使用文档

[English](./Usage.md)

面向首次安装用户的快速指南，按步骤操作即可。

## 1. 项目简介

`OpenClaw Dashboard Pro` 是一个本地仪表盘，用浏览器可视化管理本机 `openclaw`。服务启动后，大部分操作都可以在页面完成，无需再手动执行 OpenClaw 命令行：

- 安装与安装日志
- 模型配置与连通测试
- 网关启停与状态检查
- Session 查看与清理
- 更新检查与一键更新
- 技能推荐与一键安装（含 ClawHub 入口）

## 2. 环境要求

- macOS / Linux（Windows 可用 WSL）
- Node.js 18+
- 已安装 `openclaw`（若未安装，可在页面“安装”Tab 一键安装）

检查命令：

```bash
node -v
openclaw --version
```

## 3. 获取项目

```bash
cd /path/to/your/workspace
git clone <你的仓库地址> openclaw-dashboard-pro
cd openclaw-dashboard-pro
```

如果你是直接拷贝目录给朋友，可以跳过 `git clone`。

## 4. 启动服务

前台启动（推荐首次使用）：

```bash
cd /path/to/openclaw-dashboard-pro
node server.mjs
```

看到如下日志表示成功：

```text
OpenClaw Dashboard Pro (local) running at http://0.0.0.0:19190
```

## 5. 打开页面

- 本机访问：`http://127.0.0.1:19190`
- 局域网访问：`http://你的电脑IP:19190`

说明：

- 必须使用 `http`，不要用 `https`
- 若开启了代理，建议先关闭代理后再访问本地地址

## 6. 功能速览（按 Tab）

### 安装

- 一键安装 OpenClaw
- 查看安装任务状态与日志
- 更新维护：
  - `检查更新`
  - `一键更新`

### 配置

- 切换默认模型
- 模型连通性测试
- 一键接入 OpenAI Chat（Base URL / Model / API Key）
- 查看 alias / fallback

### 管理

- 网关高频操作：启动 / 重启 / 停止 / 状态
- 渠道状态
- Session 新建与清理
- 近期 Session 列表

### 优化

- 自动优化建议
- 技能推荐与一键安装
- ClawHub 技能商店入口

## 6.1 推荐流程（无需命令行）
服务启动后按以下步骤在页面完成即可：
- 在“安装”Tab 一键安装 OpenClaw
- 在“配置”Tab 配置模型并测试连通性
- 在“管理”Tab 控制网关与会话
- 在“安装”Tab 完成更新维护

## 7. 常见问题

### Q1：页面打不开

排查顺序：

1. 确认服务是否还在运行（启动终端不要关闭）
2. 确认访问地址是 `http://127.0.0.1:19190`
3. 检查 19190 端口是否监听：

```bash
lsof -nP -iTCP:19190 -sTCP:LISTEN
```

### Q2：按钮点了无响应

- 看页面底部“命令执行输出”日志
- 检查本机 `openclaw` 是否可用：`openclaw --version`
- 某些操作（更新/网关）可能需要系统权限

### Q3：局域网设备访问不到

- 确认两台设备在同一局域网
- 使用你电脑实际 IP（如 `192.168.x.x`）
- 检查系统防火墙是否拦截 19190 端口

## 8. 更新项目

```bash
cd /path/to/openclaw-dashboard-pro
git pull
```

然后重启服务：

```bash
node server.mjs
```

## 9. 停止服务

在运行 `node server.mjs` 的终端按 `Ctrl + C`。
